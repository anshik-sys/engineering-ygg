# Project Rules

## Mobile viewport work
- Only ADD new `<style>` blocks for mobile fixes. **Never edit existing CSS.**
- For mobile-specific JS behaviour, add new `<script>` blocks. **Don't touch existing scripts.**
- Revert on request — when something doesn't look right, revert just that specific change, not everything.
- Only fix what is pointed at — do not proactively change other parts of the page.
